# Building Stargate with Bazel

Here are the current supported Bazel rules. Follow the official guide to build to corresponding project in Stargate:

## Languages and frameworks
1. [Angular](https://github.com/angular/angular/blob/master/docs/BAZEL.md).
2. [TypeScript (only)](https://github.com/bazelbuild/rules_typescript).
2. [Sass](https://github.com/bazelbuild/rules_sass).
2. [Node.js](https://github.com/bazelbuild/rules_nodejs).
2. [Go](https://github.com/bazelbuild/rules_go).
2. [Python](https://github.com/bazelbuild/rules_python).
2. [Kotlin](https://github.com/bazelbuild/rules_kotlin).
2. [Scala](https://github.com/bazelbuild/rules_scala).
2. [Groovy](https://github.com/bazelbuild/rules_groovy).
2. [Rust](https://github.com/bazelbuild/rules_rust).
2. [.NET](https://github.com/bazelbuild/rules_dotnet).

## DevOps
2. [Docker](https://github.com/bazelbuild/rules_docker).
2. [Kubernetes](https://github.com/bazelbuild/rules_k8s).

## Mobile
2. [Swift](https://github.com/bazelbuild/rules_swift).
2. [Android](https://github.com/bazelbuild/rules_android).
